About

The game 'nim'

The NIM-BOY2021 is what we envision as the clash between late 90s gaming consoles and the ancient game known as 'nim'.
Nim is a simple strategy game that can be played with everything from small rocks to coins. The game is played between two people, with some number of stacks containing some number of objects each. The players then take turns picking up some number of items from whichever pile they would like. The player to empty the last pile wins the game.
