About Game

This game is a remake of a flash game named 'Beetle Field - The Forecast'. The game was made by Benjamin SoulÃ© for ludum dare 24 in 2012.
Since flash was discontinued the 31st december 2020, the original game is no longer accessible by standard web browsers. For this reason we have decided to remake the game with some quite substantial changes.
Screenshot from 'Beetle Field - The Forecast'
The game is a single player free to play strategic 2D topdown non-positional unforgiving permadeath autobattler with gambling elements. The game starts out with two teams of 3 random beetles each as well as two random upgrades. Each turn you vote for the team you think will win. If you're right you go to the next round, where you get to choose an upgrade for the team that lost. This repeats until you guess incorrectly. When this happens the game is over and you can start over.
